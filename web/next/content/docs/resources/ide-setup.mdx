## Overview

ZeroStarter includes VS Code configuration for a consistent development experience.

## Visual Studio Code

### Recommended Extensions

Defined in `.vscode/extensions.json`:

```json
{
  "recommendations": ["oxc.oxc-vscode"]
}
```

The [Oxc extension](https://marketplace.visualstudio.com/items?itemName=oxc.oxc-vscode) provides fast linting (Oxlint) and formatting (Oxfmt).

### Editor Settings

Defined in `.vscode/settings.json`:

```json
{
  "editor.defaultFormatter": "oxc.oxc-vscode",
  "editor.formatOnSave": true
}
```

## Linting & Formatting

The Oxc extension reads the project's root config files, so editor and CLI behavior match:

- `.oxlintrc.jsonc`: Oxlint rules
- `.oxfmtrc.jsonc`: Oxfmt formatting (no semicolons, sorted imports, Tailwind class sorting)

You can also run them from the terminal without VS Code:

```bash
bun run format        # format the tree with Oxfmt
bun run format:check  # check formatting without writing
bunx oxlint           # lint the tree with Oxlint
```

See [Code Quality](/docs/manage/code-quality) for how these run automatically on commit.

## Cursor

[Cursor](https://cursor.sh) uses the same VS Code configuration automatically.

Additionally, ZeroStarter includes:

- **AGENTS.md**: AI assistant instructions at repository root
- **.agents/skills/**: Structured skills for AI-assisted tasks (see [AI Skills](/docs/resources/ai-skills))
